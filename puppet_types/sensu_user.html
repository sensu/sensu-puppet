Overview

Examples:

Create a user

sensu_user { 'test':
  ensure   => 'present',
  password => 'supersecret',
  groups   => ['users'],
}

Change a user's password

sensu_user { 'test'
  ensure   => 'present',
  password => 'newpassword',
  groups   => ['users'],
}
