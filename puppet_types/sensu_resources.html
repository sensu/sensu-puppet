Resource Type: sensu_resources

Defined in:
lib/puppet/type/sensu_resources.rb

Summary

Metatype for sensu resources

Overview

Examples:

Purge unmanaged sensu_check resources

sensu_resources { 'sensu_check':
  purge => true,
}

Parameters

  • name (namevar)

    The name of the type to be managed.

  • purge (defaults to: false)

    Whether to purge unmanaged sensu resources. When set to true, this will delete any resource that is not specified in your configuration and is not autorequired by any managed resources.

    Supported values:
    • true
    • false
    • yes
    • no
